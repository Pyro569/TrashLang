# TrashLang
BrainF like language that I made for absolutely no specific reason at all.  
Commands (Currently Implemented):  
"%" Write  
"?" Denote Loop  
"+" Loop++  
">" End Loop  
"*" Print New Line  
"," Print Character Iterator

Hello World:  
`%?,,,,,,,,>%?,,,,,>%?,,,,,,,,,,,,>%?,,,,,,,,,,,,>%?,,,,,,,,,,,,,,,>%?,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,>%?,,,,,,,,,,,,,,,,,,,,,,,>%?,,,,,,,,,,,,,,,>%?,,,,,,,,,,,,,,,,,,>%?,,,,,,,,,,,,>%?,,,,>`  

Input:
`[&^#]`  
Print Recent Input:  
`[|\/]`  
Print Number Example:  
`[+++]` (Prints 3)
`[+++++][++++++][++++]` (Prints 564)
